TransactdORMPHP
===============================================================================
TransactdORMPHPはMySQL/MariaDB用　NoSQLプラグイン Transactdを使用したPHP ORMライ
ブラリです。

PHP最速のORMです。

## 詳細
ORMのインターフェースは基本的にLaravel5に近いものです。

## 実行環境
* データベース : MySQL 5.6以上 / MariaDB 5.5以上
* Transactd Plugin 3.6 以上

## 制限事項
* ORMのインターフェースはLaravel5完全互換ではありません。
